# SimpleTodoListApp #

Basic Todo app

- title of a group of tasks start with \#.
- task strats with \-.
- sub-task starts with a tab and a :.
- to mark a task as done add a (DONE) at the end of the task that you have finished

example:

\# General

\- Python practice

    : Do some online tuts
  
\- C++ practice

\- Apply to some jobs (DONE)
